# Sales Record Management System

[🚀 Run Locally](http://localhost:5000) - *Start with `python -m http.server 5000` or use Firebase hosting*

A comprehensive sales record management system built with Firebase. Features include real-time data tracking, offline support with service workers, and a responsive web interface for managing sales data efficiently.

## Quick Start

### Option 1: Simple HTTP Server
```bash
# In this directory, run:
python -m http.server 5000
```
Then open http://localhost:5000 in your browser.

### Option 2: Firebase Hosting
```bash
firebase serve
```

## Features

- Real-time data synchronization with Firebase
- Offline support with service workers
- Responsive web interface
- Sales tracking and reporting
- Data export capabilities

## Setup

1. Configure Firebase in `firebase-config.js`
2. Deploy or serve locally
3. Start tracking your sales data
